# python-task-portfolio
Python portfolio featuring 13 projects: string reversal, temperature converter, email validator, calculator, palindrome checker, guessing games, password strength meter, Fibonacci generator, file word counter, web scraper, Plotly visualizations, and automated file organizer.

## Tech Stack
- Python 3.x
- BeautifulSoup4 (Web Scraping)
- Requests (HTTP requests)
- Plotly (Interactive Visualizations)
- Pandas (Data manipulation)

## Tasks Included

### Beginner Level
1. **String Reversal** - Reverse any input string
2. **Temperature Conversion** - Convert between Celsius and Fahrenheit
3. **Email Validator** - Validate email address format
4. **Calculator Program** - Basic arithmetic operations
5. **Palindrome Checker** - Check if string reads the same backwards

### Intermediate Level
6. **Guessing Game** - Guess random number (1-100)
7. **Number Guesser** - Custom range guessing game
8. **Password Strength Checker** - Evaluate password security
9. **Fibonacci Sequence** - Generate Fibonacci series
10. **File Word Counter** - Count word occurrences in text files

### Advanced Level
11. **Web Scraper** - Extract quotes from websites using BeautifulSoup
12. **Data Visualization** - Create interactive plots with Plotly
13. **File Organizer** - Automate file sorting by extension

## Installation


### Clone the repository
git clone https://github.com/alimshaima/python-task-portfolio.git

### Navigate to directory
cd python-task-portfolio

### Install dependencies
pip install -r requirements.txt
